O projeto de desenvolvimento possui as seguintes funções extras:

Banco de dados: Firebase
SearchBar: Para procurar notícias por nome
Autenticação: Login e Register
AuthGuard: Método para redirecionar o usuário não logado caso tente acessar o Painel de Controle
Cpanel: Painel de controle próprio para cada colaborador. Nesse painel são acessíveis para deleção apenas as notícias criadas pelo mesmo
Sistema de Rating: Usuários logados podem avaliar a notícia, uma vez apenas dando like ou dislike na notícia
Coments: É possível comentar notícias
Pagination: As notícias foram paginadas a cada 5 por motivos de design, porém para 20 seria apenas uma alteração numérica no código.
SlideShow: O slideshow principal recebe um databinding em tempo real ao se clicar nos menus à direita (Featured, News, etc)
Auto detector de autor: Quando você se registra seu nome de usuário é gravado no database e sempre estará linkado à notícia.
Hospedagem: Você pode acessar o WebApplication site em https://portal-news-eab7f.firebaseapp.com/home
